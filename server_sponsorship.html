DC Emoji


Introduction

Hi, Welcome to server sponsership guidelines. If you are looking forward for advertising your server on the DC Emoji, this page will help you to know more about it.


Benefits

The app is downloaded by +200k people and is actively used by more than 25000 users. Just through advertising the app, the DC Emoji's own server is visited by 200 members weekly.


Requirements

Step 1. General Requirements:

  • - Minimum Members: None (It is not necessary)
  • - The server should NOT be NSFW
  • - The server should have friendly rules.

Step 2. Server settings:

  • - "Server Community" should be enabled: Available at Server Setting > Community > Enable Community
  • - "Server Widget" should be enabled: Available at Server Setting > Widget > Enable Server Widget

Step 3. Prepare Graphics:

  • - Server Logo: Atleast 256x256 or higher (Animated one is also accepted)
  • - Advertising banner: Atleast 960x540 and recommended aspect ratio 16:9. (Animated one is NOT accepted)

Step 4. Contact DC Emoji admin for approval

  • - Either contact via email: contact@dcemoji.com
  • - Or contact via Discord

Step 5. Payment

  • - Once admin approves your server, you are good to pay the fees.
  • - The payment method is usually done via Paypal, but you may ask the Admin for your favourable payment method.
  • - The payment is small as 5 usd for 3 weeks.
  • - Please note that other time period is NOT accepted, the advertisement will only stay for 3 weeks, after 3 weeks, you will need to go through Step 4 and Step 5 again.
  • - After every payment, the server will be placed at top of the server list. (Everytime, the new server is added top of the list)

If you have queries or issue related to this sponsorship, please contact us at contact@dcemoji.com
